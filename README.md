# 🎭 FakeDeafMute - Revenge Plugin

## الوصف
Plugin يخدع ديسكورد ليظهرك **أصم (Deaf)** و **مكتوم (Mute)** بينما أنت تسمع وتتكلم بحرية.

## الميزات
- 🔴 زر أحمر: تفعيل/إلغاء الديفن الوهمي
- 🔵 زر أزرق: تفعيل/إلغاء الميوت الوهمي
- يعمل على الجوال (Revenge)
- لا يترك أثراً في سجلات السيرفر
